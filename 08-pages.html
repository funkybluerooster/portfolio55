Applications Tab

The applications tab focuses on a specific set of predefined markets across all applications tabs.

  • The top of the page has a general intro with links that anchor down to specifc markets.

  • The primary divider separates this intro from the rest of the page and each market is divided by secondary dividers.

  • Each market has a predefined image and should only use these images. (see Applications Tab Images in this section )

This is commonly used when you don't need a table of contents, as it typically means the content for each item is brief and therefore there isn't a lot of content on the page requiring a table of contents.

This content format includes:

  • Image
  • Title
  • Description paragraph
  • CTA(s) (no table)

Grid options

This layout option can follow 2 different grid layout options. This will depend on your layout needs and how many items you have.

Option 1 layout:

This layout works will when you have a bit of content an you want to use a 2 column grid system. The image works best on the left in this case as it would be too small to put at the top of the column and look imbalanced.

Option 2 layout:

This layout works well when you want to put the items into 3 columns. The image in this case is proportional to the column width.

Other options:

As you can see that depending on the column width and content length, you could conceivably use different number of columns for the grid but these two are most commonly used.

If you decide to use a different number of columns, always remember to ensure there is a balance between image width and text width.
