Accordion Style Show Hide Content Listing

Deprecated - not to be used.

IMPORTANT: This functionality should be used only when necessary as it hides all content:

  • Content cannot be broken into separate pages or belongs to another secton,
  • Customers are interested in only one area of content – other content areas are ok to be hidden
  • Consult with an XA if it's appropriate to implement

Used in conjunction with a scrolling function to help keep users orientated on the page.

IMPORTANT: Do not use any classes within an anchor link on a page that contains the accordion script as it will cause the link to not work or work sporadically.

Script Call

To implement this, you require the following script call:

<script src="//ajax.googleapis.com/ajax/libs/jqueryui/1.9.2/jquery-ui.min.js"></script>

NOTE: The script calls needs to be placed AFTER the base jquery library call which is usually found in the header.

Define Accordion Block

Each block of content needs to be placed within a parent container div with the following markup:

<div id="UNIQUE_ID" class="accordion">
    <h3>Accordion Header</h3>
    <div>Accordion Content</div>
</div>

Anchor link to specific accordion item

If you want to have an anchor link at the top of the page, that links to a specific block of content in the accordion, the following link will link to that section and open the panel you are targeting. Add the following code to your <a> tag.

Where:

  • anchor_name_for_section = the section block where you may have a list of accordion items

  • accordion_item_num = the number in sequence, starting at 0, where the accordion item is located, this will open that particular item. For instance, you may have 5 items in one accordion. These can be targeted using numbers 0,1,2,3,4.

Additional Script

Finally, The following script needs to be added to your page AFTER your content. This allows the DOM to be loaded first.

This script requires editing: UNIQUE_ID needs to be initialized for each unique accordion. If you have only one accordion on the page, you only need to set this once.

<script>
    /* Checks for IE and turns off animation */
      if ($.browser.msie){
      var aOptions = {active: false, collapsible: true, autoHeight: false, animated: false};
    } else {
      var aOptions = {active: false, collapsible: true, autoHeight: false, animated: 275};
    }

$(function() {
    /* Initialize all items for the accordion */
    $( "#UNIQUE_ID" ).accordion(aOptions);
    $( "#UNIQUE_ID" ).accordion(aOptions);      });

$('h3').on("click", ".ui-accordion-header", function(e) {
  /* sets the panel that has been open, to the top of the screen to reorientate the visual view */
  var firstAnchor = $(e.target).find('a')[0]
  setTimeout(function(){
    $('html, body').animate({
    scrollTop: $( firstAnchor ).offset().top
    }, 275);
  }, 300);
   return false;
});
