Livesite Wrapper

Live Site Template

This is a template that mimicks a LiveSite page on TI.com. It can be used for demo's for any number of TI pages. Since this template can be used on the ww server it makes it much easier to use than the dev server to build out complex or simple demo pages to show a client. It can also be used for campainge style pages that the client wants to mimic the look of TI.com pages.

It is NOT to be used as a substitue for Live Site pages.

Unique features of this template:

  • Colapsable left menu that can be customized to mimic product tree.
  • The main content area stretches to fill the overall width when the menu is colapsed.
  • The overall width of the template can be controlled in the local CSS.

In order to use this template please copy the CSS folder, img folder, and html document and paste in your folder on the ims server or ww server. The overall width of the page can be controlled in the CSS file: .pageGroup { max-width: 1680px; }

Happy demoing!!
